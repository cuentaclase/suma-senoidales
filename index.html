Construcción de Señales mediante Sumas de Senoidales

Esta aplicación interactiva te permite comprender cómo cualquier señal puede construirse sumando varias ondas senoidales. Ajusta la amplitud, frecuencia y fase de cada señal, selecciona cuántos períodos visualizar, y observa en tiempo real cómo se combinan para formar la señal resultante.
